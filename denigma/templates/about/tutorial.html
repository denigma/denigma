Overview

Denigma is unique as everything in it is fully editable as well as programmible. Denigma can in priniciple be changes in two fashions:

  1. Content: regarding the actually content its composed of.
  2. Design: how it is conceptional built.

As soon as your Email address is verified you gain staff status and will be able to access the admin interface, hence recieve acces to the tables in Denigma db.

Data in Denigma is organized according to application (apps). There are for instance annotations, interactions, datasets, expressions and lifespan apps.:

Annotations integrates information majorly from external sources such as gene annotation databases.

Interactions is specialized on integrating heterogenous interaction types.

Expressions contains transcriptomics or proteomics profiles of gene activity measurements.

Datasets is a collection of other diverse data (e.g. from individual studies) where each table represents a single dataset.

Lifespan is most important constitute of Denigma which is dedicated to systematically decipher the aging process and reveal the limits of life. In the lifespan section the probaly most import model are describing the factors with limit lifespan.

Viewing Entries in Denigma db

To switch to the admin interface of Denigma click either on Denigm db on the left top in the hierarchy navigation or under your username in the right corner on admin. Both ways are equivallent.

You will now be confronted with a list of all the applications installed in denigma as well as all the models which represent database tables.

You can click on a given model in an application section and get a list of all entries in the table.

Each entry can be seen with all its associated information and should be fully editable.

Changing Denigma db

Denigma's core philosphy is that everything is modifyable by everyone. Denigm's Wiki and Todos already fullfil this criterion nicely. In addition all other data content can be modified in the Denigma's db admin.

Programming Denigma

Functionality to Denigma can easily be added by altering an existing app or just adding a new app.

Denigma's source code is open and its whole creation from the beginning is totally documented with version control system.

The source code of Denigma is positioned at Git Hub: Denigma Source
